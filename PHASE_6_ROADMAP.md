# Phase 6 Roadmap

## Recommended Sequence

1. Validate follow-up demand.
2. Build minimal family account only after repeated return behavior.
3. Add founder status workflow before partner portals.
4. Pilot partner referrals manually.
5. Add resource quality governance.
6. Design school/municipality pilots only after evidence of demand.

## Do Not Build Yet

- Full dashboards
- Marketplace
- Multi-tenant portals
- Paid subscriptions
- AI memory
- Knowledge synchronization
- Organization portals

## Recommended Phase 7 Scope

Private beta continuity: improve follow-up handling, founder review workflow, and manual referral preparation before adding accounts or payments.
